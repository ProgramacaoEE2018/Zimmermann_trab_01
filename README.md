Primeiramente o programa irá declarar as variáveis iniciais (notas de VC, VF, MVE)com um valor base = 0
Em seguida, pergunta-se ao operador quantas VEs foram feitas no semestre, a fim de saber quantas interações serão feitas.
Cria-se então os vetores VE, que armazena as notas, e PVE, que armazena o peso de cada VE.
Depois, a média é calculada somando os produtos das VEs com seus pesos e dividindo pelo numero de VEs.
Apos informar a média de VE, é perguntada a nota da VC para calcular quanto será preciso para a VF.
Caso a soma da MVE com a VC seja maior ou igual a 12, o aluno precisa de no máximo, tirar a nota 4 (mínima requerida pelo IME) e então ele recebe essa informação.
Caso seja menor que 12, o programa calcula quanto será necessário para o aluno terminar o semestre com nota 5 e o informa a nota de VF mínima para isso.
